# CandConnect

## Descrição do Projeto

O CandConnect é uma plataforma digital criada para fortalecer o comércio local da cidade de Candeias, conectando pequenos empreendedores aos moradores da região por meio de uma aplicação web moderna e intuitiva.

A plataforma permite que comerciantes locais cadastrem seus negócios, divulguem promoções e apresentem seus serviços, facilitando a divulgação e ampliando a visibilidade dos estabelecimentos locais.

Este projeto foi desenvolvido como simulação da primeira versão funcional de uma startup, integrando frontend e backend através de uma API REST totalmente funcional.

## Objetivo

Desenvolver uma plataforma capaz de conectar consumidores e empreendedores locais, oferecendo um ambiente digital organizado para divulgação de serviços, promoções e negócios da cidade.

## Funcionalidades

* Cadastro de negócios locais
* Cadastro de promoções e serviços
* Listagem dinâmica das informações cadastradas
* Atualização automática sem recarregar a página
* Navegação entre páginas institucionais da plataforma
* Integração completa entre frontend e backend via API REST

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Node.js
* Express
* Fetch API
